Last night I finished my onemonthrails app.  In the end, it took me around 40hrs or so to complete. I would definitely recommend the program to anyone looking to get some more experience with rails.  Previous to doing this I had tried some of Hartel's rails tutorial which is great, but seemed slightly technical for the total beginner. The omr course gave me a good basic understanding of the MVC architecture which will be key as I move forward with rails.

I think one of the great things about omr is that you build something and you build it quickly.  As a beginner it's great to see how "easy" it is to get something up and running.  After building this I am definitely motivated to get going on my next app.  For me, it makes me want to see what else I can do, what other features I can add etc.  We've only just scratched the surface with this app.
  
Check out my app here:  Let me know what you think.  [my app](http://pinteresting--omr.herokuapp.com/)
  
I know if you've seen one "pinteresting" clone you've seen them all.  I do plan on adding some extra features to the app in the future, it's on my "to do" list.  I want to possibly play around with some API's and maybe get some e-mail features set up.
  
So, now what am I going to do. I have been spending some time doing ruby koans and also ruby the hard way. My plan currently is to keep going with these while at work (lunch, breaks) and use as fill in while I start the Hartel tutorial again and work on that.  While the rails is fresh in my head, I want to start something to reinforce it.
